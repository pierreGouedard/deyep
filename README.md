## Overview

This branch host the deyep algorithm, an algorithm that can be used to perform multiple computer vision task. This algorithm 
mainly relies on the optimisation of a graph structure described in the document 

"*On Recovering Latent Factors From Sampling And Firing Graph*"


Accessible at [Arxiv web site](https://arxiv.org/abs/1909.09493). The algorithm will be tested on standard computer vision 
task using the most famous open source benchmark dataset.
